# /plans

Claude writes Plan.md files here before executing multi-step tasks.

## File Naming
`PLAN_{type}_{student_id}_{YYYY-MM-DD}.md`

## Example
`PLAN_study_session_student-001_2026-02-20.md`

## Lifecycle
inbox/ or needs_action/ → Claude reads → creates plan here → executes → moves plan to done/
